# Collections management tool

Collection management tool APIs + admin

## Requirements

* PHP >= 7.1 + extensions:
    * ctype
    * curl
    * iconv
    * json
    * mysql
    * pdo
    * xml
    * zip
* Make
* OpenSSL
* Optional (for dev env):
    * Docker
    * Docker-compose
